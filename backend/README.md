# Backend

## Getting Started

- Clone the repo: `git clone https://github.com/sanelehlabisa/roventics-site-backend.git`
- Create virtualenvironment: `python -m virtualenv .venv`
- Activate virtualenvironment: `. .venv/Scripts/activate`
- Install dependecies: `.venv/Scripts/python -m pip install -r requirements.txt`
- Run the app: `.venv/Scripts/python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload`
